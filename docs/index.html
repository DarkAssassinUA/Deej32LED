🚀 Deej32Led Installer

Вы можете прошить вашу плату ESP32 прямо из браузера.

Подключите плату по USB-кабелю и нажмите кнопку ниже (работает только в браузере платформы Chromium, как Chrome или Edge).

You can flash your ESP32 board directly from the browser.

Connect the board via USB cable and click the button below (works only in Chromium-based browsers like Chrome or Edge).

Если плата не подключается автоматически, зажмите кнопку BOOT на ESP32 при подключении кабеля. If the board does not boot into flash mode automatically, hold the BOOT button on the ESP32 while connecting the cable.
